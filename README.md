# Expense Tracker 💰

A simple Expense Tracker built using **HTML, CSS, and JavaScript**. It allows users to manage daily expenses with persistent storage using the browser's **Local Storage**.

## Features

- Add, edit, and delete expenses
- Search by description
- Filter by category
- View total expense
- Data persists using Local Storage
- Form validation

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Run Locally

Clone the repository:

```bash
git clone https://github.com/krishansah13/expense-tracker.git
```

Then open `index.html` in your browser.
